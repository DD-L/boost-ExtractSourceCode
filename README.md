# boost-ExtractSourceCode

* win32
    1. Makefile.win32
        ```shell
        cmd /c make -f Makefile.win32
        cmd /c make clean -f Makefile.win32
        ```
    2. boost_init.bat
        ```shell
        cmd /c boost_init.bat
        ```
* linux
    1. Makefile
        ```shell
        $ make
        $ # or make boost 
        $ # or make boost.force
        $ make clean 
        ```
    2. boost_init.sh
        ```shell
        $ sh boost_init.sh --help
        ```
